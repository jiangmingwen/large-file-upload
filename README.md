# BIG-FILE-UPLOAD 

实现大文件切片上传，文件秒传，断点续传

## Features

 * 客户端采用vue 2.x，采用XMLHttpRequest封装接口，利用web worker建立子线程进行文件切片
 * 服务端采用nodejs

### Getting Started

- #### Client:

  ```
  cd web && npm install
  
  npm run serve
  ```


- Server

  ```
  cd serve && npm install
  
  npm start
  ```
